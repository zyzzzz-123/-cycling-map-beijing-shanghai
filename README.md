# 清华大学到上海浦发有家申鸿苑骑行计划地图

这是一个本地静态页面，用在线地图展示清华大学到上海浦发有家申鸿苑、经过滕州市的 12 天骑行计划。

## 使用

1. 打开 `index.html`，或在本目录启动一个静态服务器。
2. 页面会自动加载地图。

页面会先显示人工校准的分日路线；地图加载后，会尝试按每天的具体途经点分段规划。

如果页面提示地图加载失败，优先检查本机代理或网络连接后刷新。

## 路线口径

- 山东南段按“济宁-邹城/微山北-滕州-枣庄/峄城外缘-徐州外缘”处理，满足经过滕州市，但比原台儿庄运河走廊更起伏。
- 总时间控制为 12 个骑行日，属于高强度压缩版；D2-D7 连续多天风险偏高。
- 每日爬升是按地形带和城市节点估算的累计爬升区间，不等同于码表实测值。
- “主要路线”优先写国省道走廊和可替代的辅路/县乡路；出发当天仍要用高德骑行导航逐段排查施工、禁行桥、高架匝道和大车路段。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库，例如 `cycling-map`。
2. 把本目录所有文件推送到仓库根目录。
3. 打开仓库 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`，保存。
6. 等待 GitHub Pages 部署完成，访问页面给出的 URL。
